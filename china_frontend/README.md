## Correr Localmente

Clonar proyecto

```bash
  git clone https://github.com/NataliaLopezO/frontend_proyecto_integrador_china.git
```



Instalar dependencias

```bash
  npm install
```

Correr el servidor

```bash
  npm run dev
```


## Correr Pruebas

Para realizar pruebas, utilizar el siguiente comando

```bash
  npm run test
```
